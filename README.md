## Install

Get it with **npm**:

```sh
npm install -g ibrew
```

## Usage

Search Homebrew packages via ibrew with `ibrew <searchterm>`:

```
ibrew say
```

**OR**

bypass **npm install** and run it at once using **npx**:

```sh
npx ibrew say
```

## Requirements

- macOS
- [Homebrew](https://brew.sh)
- Node.js
